Version history of rPE - rEdoX Packet Editor
============================================

Below you find a list with the added features, changes and fixes for 
each version.

Version 1.4.2.0 (Sep 25 2007)
-----------------------------------
- Type Search Added
- "Save as html" added

Version 1.4.1.0 (Sep 18 2007)
-----------------------------------
- MDI Minimize/Maximize fix
- Docking Added
- Conversion Added

Version 1.4.0.0 (Sep 8 2007)
-----------------------------------
- Heavy GUI changes
- IPC Method changed
- Crashes fixed
- GetSysCallNr.exe -> Heavy Speed increase (3m to <5s)


Version 1.3.0.1 (Jun 24 2007)
-----------------------------------
- MDI-Focus fix
- OpenProcess-Dialog fix


Version 1.3.0.0 (Jun 23 2007)
-----------------------------------
- Kernelmode support added
	- Kernelmode Stealth
	- Kernelmode based DLL injection
- GetSysCallNr.exe / Target.exe
- Antidetections for rPE.dll added
- GUI fix
- Fixed MDI-AV


Version 1.2.0.0 (Jun 10 2007)
-----------------------------------
- Disassembly View
- Syntax highlighting for the disassembly view
- Standalone Filter
- Ignore list
- each packet got his own Socket ID 
- Little Hotfixes
- Example Customfilter
- CreateRemoteThread injection
- SetThreadContext injection


Version 1.1.0.0 (Nov 19 2006)

- Bytes beim senden erhoehen(so wie bei wpe)

- Kleine bugfixes
- INI-support


Version 1.0.0.0 (Oct 10 2006)
