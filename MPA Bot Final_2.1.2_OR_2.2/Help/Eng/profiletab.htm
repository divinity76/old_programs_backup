Profiles - Profile tool will save and load all things on the program (i.e text boxes, check boxes). After you load bot will be automaticly disabled because some of bad people. Some of them may send you a profile with "Rainbow Outfit" enabled and when you load it then it will automaticly turn on "Rainbow Outfit". That's why i did turning the bot off after loading profile.

1. If you want to save profile (bot) then write in the text box below the list the name of profile (i.e "Keth's Profile"). After it click "Save Current" and done. When you close your bot later you can load it your profile.

2. If you want to load profile (bot) then select from the list profile you want to load and click "Load Selected". After it bot will be disabled. If you want to know why the bot is disabling then read read step one.

3. If you want to delete profile then select from the list profile you want to delete and click "Delete Profile". After you delete it you can CAN get it back from your dustbin.

Remember! Lists like Cavebot list, Auto Responder list, Safe List will be NOT saved. They can be saved using their special Save Button. The button should be near the list.
