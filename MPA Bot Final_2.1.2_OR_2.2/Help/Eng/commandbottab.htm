Commands - This list will show you the avaible commands. You can use them by writing it in Tibia Window same as normal message.

1. *MPA Bot*Light: #

Replace # to a number between 0 and 17. The number means size of the light.

2. *MPA Bot*Light: Off

This command will turn off light forever until you will change light level.

3. *MPA Bot*Light: None

This command will come back to the real Tibian light without any fake light.

4. *MPA Bot*Light: Torch

This command will make light near you with size of Torch.

5. *MPA Bot*Light: Full

This command will turn on full light near you with maximum size.

6. *MPA Bot*Exp

This command will show you in Status Bar how much you need experience to next level.

7. *MPA Bot*Add Waypoint

This command will add your current position (waypoint) to cavebot waypoint list.

8. *MPA Bot*Delete Last Waypoint

This command will delete last added position (waypoint) in the cavebot waypoint list.

9. *MPA Bot*Delete Last Waypoint

This command will delete last added position (waypoint) in the cavebot waypoint list.

10. *MPA Bot*#1 Creature: #2

Replace #1 to the name of the rune you want to shot and #2 to creature you want to shot with #1 rune. You can also replace #1 to %targetname or %selfname and it will shot rune #1 to a player you are currently attacking. Here are some examples:

*MPA Bot*SD Creature: Bubble - Use Sudden Death Rune on the creature Bubble.

*MPA Bot*UH Creature: Keth - Use Ultimate Healing Rune on the creature Keth.

*MPA Bot*UH Creature: %selfname - Use Ultimate Healing Rune on your character.

*MPA Bot*EXP Creature: %targetname - Use Explsion Rune on creature you are currently attacking.

*MPA Bot*HMM Creature: Dragon - Use Heavy Magic Missile on creature Dragon.

You can combine above examples of commands or even add them to Tibia Hotkeys with Auto Send. Please remember that: Rune that you are using (#1) must be opened and visible on the screen.
